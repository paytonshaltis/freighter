    Basic Usage

    Learn the basics of Freighter carousels and start creating your own. Start by selecting a container to house your carousel. Select methods for resizing and wrapping your carousel, then begin adding items. Add and delete items or change the properties of your carousel at any time.

    Creating a Carousel

    Freighter carousels are highly customizable, require only three properties in order to create the simplest of carousels:

    • Container - The container that houses the carousel. This is created by you, and the ID is passed to the carousel constructor.
    • Rezise Method - How the carousel will behave when the window is resized. These range from scaling the size of carousel content to decreasing the number of visible carousel items.
    • Wrap Method - How the carousel items will 'wrap' when scrolling past the end of the carousel. A carousel can wrap around infinitely, or prevent scrolling when one end is reached.

    The three required carousel properties will be passed to the Freighter constructor inside of a CarouselOptions object. By the time all required carousel properties have been discussed on this page, a call to the constructor will look something like this: 

    
      const myCarousel = new Freighter({
        containerID: 'my-carousel',
        resizingMethod: 'stretch-scale',
        wrappingMethod: 'wrap-smart'
      });

    Choose a Container

    Any div that you create can be used to house a Freighter carousel. The only requirement is that you assign the div a unique ID that can be used to locate and modify this DOM element: 

        
      <div id="my-carousel">
        <!-- Carousel items will eventually go in here! -->
      </div>

    You can add reposition and style this container. All Freighter carousel DOM elements will be appended to it (they will be made the children of the chosen div). The classes and ID of the chosen container will not be altered, so feel free to apply custom CSS classes to your container.

    Pick a Resize

    The next required property for any carousel is the resizing method. When the width of the browser increases or decreases, the carousel must respond in an appropriate way. The way in which the carousel responds can be configured throught the resizingMethod property. One method included in Freighter is stretch-scale, which will stretch or shrink the carousel while preserving the aspect ratio of each item.

    You can see the full list of resizing methods here, as well as a variety of examples that show off how each resizing method can be used. For the purposes of this page, the stretch-scale value will be used.

    Try resizing the browser window and see how the carousel responds. You will notice a few things:

    • The number of visible items in the carousel remains the same.
    • The aspect ratio of each item is preserved, even though each item's width and height increases or decreases with the size of the browser window.
    • The size of the gap between carousel items is preserved.
